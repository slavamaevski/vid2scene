Responsive Embed

Instructions

To embed the vid2scene viewer on your website, use the following iframe code:

            <iframe src="https://vid2scene.com/viewer/{{ spj_id }}/" 
                    width="800" 
                    height="600" 
                    style="border: none;" 
                    allowfullscreen>
            </iframe>

Replace `{{ spj_id }}` with the actual Scene Processing Job ID you want to embed.

Responsive Embed Code Example

<div style="position: relative; width: 100%; height: 0; padding-bottom: 75%;">
    <iframe src="https://vid2scene.com/viewer/{{ spj_id }}/" 
            style="position: absolute; top:0; left: 0; width: 100%; height: 100%;" 
            frameborder="0" 
            allowfullscreen 
            sandbox="allow-scripts allow-same-origin">
    </iframe>
</div>

This code ensures that the embedded viewer is responsive and maintains the correct aspect ratio across different devices.
