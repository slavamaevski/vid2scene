3DGS AutoLOD

Generate LOD versions of 3D Gaussian Splatting PLY files in your browser.

1. Select Input File

No file selected

Note: Need to process a larger scene? Use the 64-bit build which supports files larger than 1GB (slightly slower, Chrome/Firefox only. Firefox works best).
For native performance and commercial use, contact {{ contact_email }} for the native binary.

2. LOD Settings

Use Percentage to reduce to a fraction of the original (e.g. 50%), or Absolute Count to target a specific number of gaussians. It will save a separate PLY file for each comma separated value.

Comma-separated percentages (e.g., 50,25,10 = keep 50%, 25%, 10%)
